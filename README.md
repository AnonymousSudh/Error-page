## Run Locally

Clone the project

```bash
  git clone https://github.com/AnonymousSudh/Error-page.git
```

Go to the project directory

```bash
  cd errorpage
```

Install dependencies

```bash
  npm install
```

Start the client

```bash
  npm start
```
